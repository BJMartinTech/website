Cartridge Support

Frequently Asked Questions


I'm not able to create more than 10 games.

Cartridge only allows tracking up to 10 games if you have not purchased Cartridge Premium.
If you'd like to track your entire video game backlog, please consider purchasing Cartridge Premium to help support us.
If you want to remain as a Cartridge Free user, and want to track new games, you can still delete older games to add newer ones, as long as you're under the 10 game limit.


I want to use the camera to take a photo of my games, but it's not letting me.

Cartridge requires iOS permission in order to be able to access the camera. You should have been asked for permission the first time you tapped "Take Photo" when editing your Game Cover Image. If you're currently not able to access the camera, it's likely you denied permission a while back. To re-enable Camera access, go to the built-in iOS Settings app, then tap Privacy, Camera, and toggle on Cartridge.
If you have given Cartridge permission, but are still not able to access the camera within the app, please contact us via email and we'll take a look!


I'm trying to search IGDB for game cover images, but none are showing up.

We use a third-party service called IGDB to search for video game covers. For best results, try to search for the exact names of a game. For example, if you're looking for "The Legend of Zelda: Breath of the Wild", try using "Breath of the Wild" or even "The Legend of Zelda: Breath of the Wild". If the full name of the game doesn't yield any results, try searching by unique keywords. For example, if you're searching for "Animal Crossing New Horizons", you can try "New Horizons".
If you're getting no results whatsoever, it's possible that the IGDB service is down. If so, please try again at a later time.
If you are not able to get any kind of results after trying the above, please contact us via email and we'll take a look!


I'm trying to purchase Cartridge Premium, but I'm not able to.

Make sure that you're logged into an Apple account on your iPhone or iPad, and that you have In-App Purchases enabled.
To check if you have In-App Purchases enabled, go to the iOS Settings app, Screen Time, Content & Privacy Restrictions. Make sure that iTunes & App Store Purchases are allowed.
If you are still not able to purchase Cartridge Premium after ensuring the above, please contact us via email and we'll take a look!
